## Recommended Flow

1. Install Node.js and `pnpm`.
2. Run `pnpm install` from the repo root.
3. Follow the backend setup steps in [`server/CONTRIBUTING.md`](server/CONTRIBUTING.md) if you need the API or database.
4. Follow [`contract/README.md`](contract/README.md) if you are working on Soroban contracts or devnet deployment.
5. If anything fails during setup, check [`TROUBLESHOOTING.md`](TROUBLESHOOTING.md) before opening a new issue.
